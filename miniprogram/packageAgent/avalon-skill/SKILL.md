## 用户意图分流

### 分支1：用户想了解阿瓦隆规则
用户说"阿瓦隆怎么玩""规则是什么""有哪些角色"
→ 调用 **showRules** API 导航到 pages/rules/rules，展示角色能力、夜间流程和胜利条件。
   可选携带 players/mordred/oberon 参数以展示对应人数的配置规则。

### 分支2：用户想找桌游主持工具
用户说"聚会需要一个主持人""有没有自动带队的""线下玩阿瓦隆"
→ 调用 **startGame** API（默认参数）进入设置页，让用户选择玩家人数和角色后开始主持。

### 分支3：用户已准备好开始游戏
用户说"阿瓦隆7人局""来一局阿瓦隆""开一局"
→ 调用 **startGame** API，传入解析出的 players（如7），进入设置页确认配置后开始夜间环节语音引导。

### 分支4：用户指定角色变体
用户说"10人局带莫德雷德奥伯伦""7人局莫德雷德"
→ 调用 **startGame** API，传入 players=10, mordred=true, oberon=true 等参数。

### 分支5：用户询问特定角色能力
用户说"梅林能看见谁""刺客怎么玩""派西维尔看谁"
→ 调用 **getRoleInfo** API，传入解析出的 roleName，返回该角色的阵营和能力说明。

## API 参考

本 Skill 提供三个原子接口：

| API | 用途 | 关键参数 |
|-----|------|----------|
| `startGame` | 配置并开始新游戏 | players(5-10), mordred, oberon |
| `showRules` | 查看游戏规则 | players(可选), mordred(可选), oberon(可选) |
| `getRoleInfo` | 查询角色能力 | roleName(必填) |

## 业务约束
- 主持过程中自动播放语音指令，用户也可手动点击「下一步」
- 暂停环节默认等待5秒，可在设置页调整1-60秒
- 7-9人局莫德雷德和奥伯伦互斥（二选一），10人局可同时选择
- 5-6人局不支持莫德雷德/奥伯伦变体（startGame 会自动忽略这些参数）
- 规则查询优先调用 showRules；游戏配置优先调用 startGame；角色咨询调用 getRoleInfo
